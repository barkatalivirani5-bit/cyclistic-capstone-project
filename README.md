## 🔍 Analysis Overview

The goal of this project was to analyze how annual members and casual riders use Cyclistic bikes differently and provide recommendations for converting casual riders to annual members.

## 📝 Key Findings

1. **Usage Patterns**: Casual riders peak on weekends, members on weekdays
2. **Ride Duration**: Casual riders take longer rides (25-30 min) vs members (12-15 min)
3. **Seasonal Trends**: Both user types increase in summer; casual riders drop more in winter
